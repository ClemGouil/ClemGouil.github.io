Meteor Shooter

Description du projet :

Au cours de ma première année à l'ENSEEIHT, lors de l'apprentissage de la programmation orientée objet, j'ai participé à un projet où nous devions développer un jeu ou une application en Java. Au sein d'une équipe de 4 personnes, nous avons décidé de produire un jeu populaire de type "shoot them up". Le but de notre jeu etait d'incarner un vaisseau, d'esquiver les météorites et de tirer sur les vaisseaux ennemis. Il comporte un mode histoire et un mode infini. Le jeu a été créé grâce à la bibliothèque JavaFX pour les aspects visuels.

Technologies utilisées :

Java JavaFx

Space Survivor

Description du projet :

Lors de ma deuxième année, pendant mon semestre à l'étranger, j'ai eu la chance de participer à un projet de développement d'une application mobile pour un jeu. Nous avons commencé l'aventure à 4, mais nous sommes rapidement tombés à 2 à la suite d'abandons. Malgré cela, nous avons réussi à créer une application mobile avec de nombreuses fonctionnalités, telles qu'un système de connexion, un système d'achat, un forum, des notifications, etc., ainsi qu'un jeu en 2D inspiré du tutoriel Roguelike 2D, dans lequel on incarne un personnage à bord d'une navette spatiale et où l'on doit passer différents niveaux au sein d'une navette remplie d'ennemis. Nous avons dû mettre en place à la fois le backend et le frontend. Pour le frontend, l'application Android a été développée avec Android Studio. Pour le backend, nous avons mis en place une API REST afin de fournir les différents services web en communication directe avec une base de données MariaDB où nous stockions l'ensemble de nos données. Le jeu a été développé avec Unity.

Technologies utilisées :

Java REST API Android Studio Unity MariaDB MySQL

The Brave Farmer

Description du projet :

Ce projet est un projet personnel sur lequel je travaille actuellement. Cela fait six mois que je me familiarise avec l'environnement d'Unreal Engine 5, un moteur de jeu très puissant. J'ai suivi de nombreux tutoriels disponibles sur YouTube, notamment ceux de Ali El Zoheiry (https://www.youtube.com/@AliElZoheiry) et Unreal Sensei(https://www.youtube.com/@UnrealSensei). Ce projet combine et regroupe plusieurs travaux issus de ces tutoriels. J'ai notamment pu découvrir l'utilisation des différents panneaux et interfaces, des behavior trees, de blueprint, la création d'environnements, etc. Le jeu consiste à incarner un personnage évoluant dans un environnement médiéval, partant à la rescousse d'un autre personnage et devant affronter des groupes d'ennemis.

Technologies utilisées :

Unreal Engine 5 Blueprint
